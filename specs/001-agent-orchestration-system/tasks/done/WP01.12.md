# WP01.12: Create .env.example with required environment variables

## Objective

Create an example environment file documenting all required and optional environment variables with descriptions and example values.

## File Paths

- `.env.example`

## Work Content

1. Create .env.example with all environment variables:
   ```
   # Database Configuration
   DATABASE_URL=postgresql://user:password@localhost:5432/agent_orchestration
   DATABASE_MAX_CONNECTIONS=20

   # Anthropic API
   ANTHROPIC_API_KEY=sk-ant-xxxxx

   # Application
   NODE_ENV=development
   PORT=3000
   LOG_LEVEL=info

   # Optional: Git Configuration
   GIT_REPO_PATH=/path/to/repo
   ```
2. Add comments explaining each variable
3. Use placeholder values (not real credentials)
4. Group variables by category

## Acceptance Criteria

- [ ] .env.example includes all required variables
- [ ] Variables grouped by category (Database, API, App)
- [ ] Comments explain purpose of each variable
- [ ] Placeholder values used (no real credentials)
- [ ] Format matches actual .env file structure

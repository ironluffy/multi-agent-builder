# WP02.14: Create Message model interface [P]

## Objective

Define the TypeScript interface for the Message model, supporting asynchronous message passing between agents with priority and status tracking.

## File Paths

- `src/models/Message.ts`

## Work Content

1. Create Message.ts with interface definition:
   ```typescript
   export enum MessageStatus {
     PENDING = 'pending',
     DELIVERED = 'delivered',
     PROCESSED = 'processed',
     FAILED = 'failed'
   }

   export interface Message {
     id: string;
     sender_id: string;
     recipient_id: string;
     content: string;
     priority: number;
     status: MessageStatus;
     created_at: Date;
     delivered_at?: Date;
     processed_at?: Date;
     metadata?: Record<string, any>;
   }

   export interface SendMessageInput {
     sender_id: string;
     recipient_id: string;
     content: string;
     priority?: number;
     metadata?: Record<string, any>;
   }
   ```
2. Add JSDoc documentation
3. Export type guards for status validation
4. Export helper types for message operations
